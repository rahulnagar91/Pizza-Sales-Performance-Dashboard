# 🍕 Pizza Sales Analysis (SQL + Power BI)

I built this project to analyze a pizza shop's sales data for a full year. I used SQL to extract the metrics and Power BI to create an interactive dashboard showing peak hours, top-selling pizzas, and revenue trends.

## 🛠️ Tools Used
* **SQL** (Data preparation)
* **Power BI** (Data modeling & Visualization)

## 📌 Key Metrics 
* **Total Revenue:** $764.45K[cite: 1]
* **Total Orders:** 19,962[cite: 1]
* **Average Order Value:** $38.30[cite: 1]
* **Average Pizzas per Order:** 2.32[cite: 1]

## 💡 Top Insights
* **Busiest Day:** Friday is the busiest day of the week with 3.3K orders[cite: 1].
* **Best Months:** July and January see the highest order volumes[cite: 1].
* **Best Seller:** The **Classic** category drives the most sales[cite: 1], and **Large** is the most popular size (45.85% of sales)[cite: 1].
* **Top Pizza:** The **Thai Chicken Pizza** generates the highest revenue ($43K)[cite: 1].
* **Worst Pizza:** The **Brie Carre Pizza** ranks lowest in revenue ($12K) and total orders[cite: 1].

## 🚀 How to View
1. Download the `.pbix` file from this repository.
2. Open it in **Power BI Desktop** to click through the interactive filters.
3. Check out the `SQL_Scripts` folder to see the queries used.
